# SwipeClean 📱

Минималистичное Android-приложение для быстрой очистки галереи свайпами (как в Tinder).

## Фичи

- **Свайп влево** — удалить фото (Android покажет системное подтверждение удаления — это безопасно)
- **Свайп вправо** — оставить фото
- **Случайный порядок** — каждая сессия показывает рандомные фото из галереи
- **Виджет на рабочий стол** — показывает случайную фотку, тап открывает приложение
- **Material 3 + авто тёмная/светлая тема** — подстраивается под систему
- **Счётчик** — сколько удалил/оставил в текущей сессии

## Как получить APK за 5 минут

### Способ 1: GitHub Actions (рекомендую)

1. Создай на GitHub новый репозиторий (например `swipeclean`)
2. Загрузи все эти файлы в репозиторий:
   ```bash
   cd SwipeClean
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/ТВОЙ_ЛОГИН/swipeclean.git
   git push -u origin main
   ```
3. Открой вкладку **Actions** в репозитории — workflow запустится автоматически
4. Через ~3-5 минут перейди в раздел **Releases** — там будет готовый `app-debug.apk`
5. Скачай APK на телефон и установи (разреши «Установка из неизвестных источников»)

### Способ 2: Локально через Android Studio

1. Открой папку `SwipeClean` в Android Studio
2. Дождись синхронизации Gradle
3. `Build → Build Bundle(s) / APK(s) → Build APK(s)`
4. APK появится в `app/build/outputs/apk/debug/`

## Установка виджета

После установки приложения:
1. Долгое нажатие на пустое место на рабочем столе
2. «Виджеты» → найди **SwipeClean**
3. Перетащи на рабочий стол
4. Тап по виджету открывает приложение, тап на `↻` обновляет фото

## Важные замечания

- **Требуется Android 8.0+** (API 26)
- При первом запуске попросит разрешение на чтение галереи
- При удалении на Android 11+ система покажет подтверждение — это защита от случайного удаления, так устроено API. Обойти нельзя (и не нужно).
- Удалённые фото попадают в **корзину Галереи** (на большинстве устройств можно восстановить в течение 30 дней)
